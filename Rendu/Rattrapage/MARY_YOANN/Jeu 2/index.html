Color Path

Gameplay :

Déplacer les ronds de couleur dans la case correspondante à sa couleur. Si la case choisie est de mauvaise couleur alors le joueur perd une vie. Si le joueur n'a plus de vie alors il recommence à 0. Appuyer sur R pour recréer un rond, touches fléchées pour se déplacer

Cours : Creative Coding, Max Vermesch & Alexandre Rivaux, e-art sup 2018
