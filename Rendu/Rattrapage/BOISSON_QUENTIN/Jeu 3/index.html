Letter Sorting

Gameplay :

Des lettres arrivent depuis la gauche de l'écran, il faut les trier selon leur type : voyelle ou consonne. Il faut appuyer sur Flèche Haut ou Flèche bas pour les alligner dans le bon "couloir". Si le mauvais choix est fait alors une vie est perdue, sinon le score augmente.

Cours : Creative Coding, Max Vermesch & Alexandre Rivaux, e-art sup 2018
